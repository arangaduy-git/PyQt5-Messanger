1. Название проекта - PyQt5 Messanger
2. Автор проекта - Егор Чешуин
3. Описание идеи - Чат с регистрацией и со своим сервером, где люди могут вместе общаться.
4. Описание реализации - При запуске программы надо войти в аккаунт, после чего настраиваем IP адрес и порт, подключаемся и общаемся. Основые классы: Chat, Registration, Login, Server
5. Интерфейс написан на PyQt5, для сервера используется socket, связь с БД с помощью sqlite3.
6. Скриншоты в этой папке
